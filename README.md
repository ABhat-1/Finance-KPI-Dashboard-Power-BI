This project is a end-to-end Finance KPI Dashboard developed using Microsoft Power BI. It focuses on tracking key financial performance indicators and turning raw financial data into clear, actionable insights within a compact, interactive dashboard.

📊 Project Overview

The goal of this project is to design a professional finance dashboard that helps monitor core KPIs such as revenue, expenses, profit, and margins. It provides a high-level financial snapshot along with detailed breakdowns for better decision-making.

⚙️ Key Features

1. Financial KPI tracking (Revenue, Profit, Expenses, Net Margin)
2. Month-over-month performance analysis
3. Trend visualization for income and costs
4. Category-wise or segment-wise financial breakdown
5. Interactive filters for time periods and dimensions

🧹 Data Preparation

1. Imported financial dataset (Excel/CSV)
2. Cleaned and transformed data using Power Query
3. Structured data for analysis (dates, categories, values)
4. Ensured proper data types and consistency

📐 Data Modeling & DAX

1. Built relationships between financial tables
2. Created DAX measures such as:
a. Total Revenue
b. Total Expenses
c. Net Profit
d. Profit Margin %
e. Year-to-Date (YTD) / Month-to-Date (MTD) metrics

📈 Visualizations

1. KPI cards for high-level financial metrics
2. Line charts for revenue and expense trends
3. Bar charts for category comparisons
4. Time intelligence visuals (monthly/quarterly performance)
5. Slicers for dynamic filtering

💡 Key Insights

1. Identification of profit trends over time
2. Expense behavior analysis and cost drivers
3. Revenue growth tracking across periods
4. Overall financial health assessment at a glance

🚀 Tools Used

1. Microsoft Power BI
2. Excel / CSV financial datasets
3. DAX (Data Analysis Expressions)
4. Power Query Editor
